# ReactStarter

#### A template for my new React projects, early work in progress.

Currently this starter is using
* React
* redux
* react-redux
* normalize.css
* redux-promise
* image-webpack-loader

For workflow i am using
* webpack
* babel
* style-loader
* sass-loader
* file-loader
* jsx-loader
* html-webpack-plugin

For DEV only
* webpack-dev-server

For Production
* node
* express
